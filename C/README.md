# Motor-Control-Stm32 
This file directory containing stm32 projec using C language
# STM32 BOARD PINS

ANGEL SENSOR:                 
SCL     ---> PB13
SDA     ---> PB11
VSS     ---> 3.3V
Ground  ---> GND 

USART:                 
TXD     ---> PC4
RXD     ---> PC5
VSS     ---> 3.3V
Ground  ---> GND

MOTOR DRIVER:
ENCA    ---> PB5
ENCB    ---> PB4
ENABLE  ---> PA4
IN1     ---> PA9
IN2     ---> PA10
3.3V    ---> 3.3V
5.0V    ---> 5.0V
Ground  ---> GND
